# Social Media Console App 🖥️💬

A **mini social media system** built in **C#** as a console application.  
Practice project for learning **OOP, lists, and basic CRUD operations** in C#.  

---

## Features

- Register / login users  
- Create posts  
- Like posts  
- Comment on posts  
- View all posts with likes and comments  

---

## Tech Stack

- **C#** – programming language  
- **.NET Console App** – runtime environment  
- In-memory storage (Lists)
